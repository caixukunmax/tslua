# TS-Skynet 快速命令参考

## 🚀 最简单用法

```bash
./start.sh              # 打开菜单界面（推荐！）
```

## 📋 快捷命令

| 命令 | 说明 |
|------|------|
| `./start.sh` | 打开菜单界面（什么都有） |
| `./start.sh build` | 编译 TypeScript → Lua |
| `./start.sh status` | 查看服务状态 |
| `./start.sh stop` | 停止服务 |
| `./start.sh restart` | 重启服务 |
| `./start.sh hotfix login` | 热更新登录服务 |
| `./start.sh logs` | 查看日志 |
| `./start.sh node` | Node.js 模式（开发用） |

## 📝 完整命令参考

### 开发调试
```bash
./start.sh              # 菜单界面
./start.sh node         # Node.js 模式运行
./start.sh build        # 编译 TS→Lua
./start.sh hotfix all   # 热更新所有服务
```

### 服务管理
```bash
./start.sh start        # 启动服务（菜单）
./start.sh status       # 查看状态
./start.sh stop         # 停止服务
./start.sh restart      # 重启服务
./start.sh logs         # 查看日志
```

### 维护
```bash
./start.sh clean        # 清理构建产物
```

## 🔧 NPM 命令

```bash
npm run dev             # Node.js 开发模式
npm run build:ts        # 编译 TS→Lua
npm run build:proto     # 编译 Protobuf
npm run server:start    # 启动 Skynet 服务
npm run server:stop     # 停止服务
npm run hotfix          # 热更新服务
```

## 📁 重要路径

| 路径 | 说明 |
|------|------|
| `src/` | TypeScript 源码 |
| `dist/lua/` | 编译后的 Lua 代码 |
| `config/` | 配置文件 |
| `docs/` | 文档 |
| `logs/` | 运行日志 |

## 💡 常用场景

### 1. 日常开发
```bash

```


### 3. 查看服务状态
```bash

```

### 4. 停止服务
