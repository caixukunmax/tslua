## 🔄 数据流向

### 1. Protocol Buffers 流程
```
protocols/                    # .proto 源文件
    │
    │ build_proto.sh
    ▼
src/protos/                   # 生成的 TypeScript
dist/lua/protos/              # 生成的 .desc 描述文件
```

### 2. Luban 配置表流程
```
config/tables/
├── defines/                  # XML 定义
└── datas/                    # Excel 数据
    │
    │ build_tables.sh
    │ (dotnet Luban.dll)
    ▼
src/tables/                   # 生成的 Lua/TypeScript 代码
```

### 3. TypeScript → Lua 流程
```
src/                          # TypeScript 源码
    │
    │ npm run build:ts
    │ (tstl --project config/tsconfig.lua.json)
    ▼
dist/lua/                     # Lua 代码
    │
    │ Skynet 读取
    ▼
Skynet 运行
```

### 4. TypeScript → Node.js 流程
```
src/                          # TypeScript 源码
    │
    │ tsc --project config/tsconfig.json
    ▼
dist/nodejs/                  # Node.js 代码
    │
    │ Node.js 运行
    ▼
Node.js 开发调试
```

## 📋 关键文件说明

| 文件/目录 | 说明 |
|-----------|------|
| `protocols/*.proto` | Protocol Buffers 源文件 |
| `src/protos/proto.ts` | 手写的 Proto 运行时（TypeScript） |
| `dist/lua/protos/*.desc` | Protobuf 描述文件（Skynet 用） |
| `config/tables/luban.conf` | Luban 配置文件 |
| `config/tables/defines/*.xml` | 配置表 XML 定义 |
| `config/tables/datas/*.xlsx` | Excel 配置数据 |
| `src/tables/` | Luban 生成的配置表代码 |
| `tools/luban/Luban/` | Luban 工具 DLL |
| `config/tsconfig.lua.json` | TSTL 编译配置 |
| `config/tsconfig.json` | TypeScript 编译配置 |
| `skynet/config.tslua` | Skynet 运行时配置 |
| `start.sh` | 快速启动脚本 |
| `scripts/start.sh` | 菜单式启动脚本 |
