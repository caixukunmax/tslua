# 环境安装指南

本文档详细说明如何在 Windows、Linux、macOS 上安装 TS-Skynet 所需的所有依赖。

---

## 📋 依赖概览

| 依赖 | 用途 | Windows | Linux | macOS |
|------|------|---------|-------|-------|
| **Node.js 20+** | 运行 CLI 和开发调试 | ✅ 必需 | ✅ 必需 | ✅ 必需 |
| **.NET 8.0 SDK** | Luban 配置表编译 | ✅ 必需 | ⚠️ 可选 | ⚠️ 可选 |
| **Docker + WSL2** | Skynet 生产部署 | ✅ 必需 | ✅ 必需 | ✅ 必需 |

> 💡 **提示**: 
> - Windows 用户需要安装所有三项
> - Linux/macOS 用户如果不需要编译 Luban 配置表，可以跳过 .NET SDK

---

## 🪟 Windows 环境安装

### 1. 安装 Node.js 20+

**步骤**:
1. 访问 https://nodejs.org/
2. 下载 LTS 版本（20.x 或更高）
3. 运行安装程序
4. 验证安装：
   ```bash
   node --version      # 应显示 v20.x.x
   npm --version       # 应显示 9.x.x 或更高
   ```

### 2. 安装 .NET 8.0 SDK ⭐

**为什么需要**: Luban 配置表工具使用 .NET 8.0 编译

**步骤**:
1. 访问 https://dotnet.microsoft.com/download/dotnet/8.0
2. 下载 **.NET SDK 8.0.x** (不是 Runtime)
   - 选择 x64 版本（除非你使用 ARM 设备）
3. 运行安装程序
4. 重启终端（重要！让 PATH 生效）
5. 验证安装：
   ```bash
   dotnet --version    # 应显示 8.0.x
   ```

> ⚠️ **常见错误**: 
> - 只安装了 Runtime 而不是 SDK → 确保下载的是 "SDK" 不是 "Runtime"
> - 安装后 `dotnet` 命令找不到 → 重启终端或重新打开 PowerShell

### 3. 启用 WSL2

**为什么需要**: Docker Desktop 在 Windows 上需要 WSL2 才能正常运行

**步骤**:
1. 以**管理员身份**打开 PowerShell
2. 执行命令：
   ```powershell
   wsl --install
   wsl --set-default-version 2
   ```
3. 重启电脑（如果需要）
4. 验证安装：
   ```bash
   wsl --list --verbose
   ```
   应看到类似输出：
   ```
     NAME      STATE           VERSION
   * Ubuntu    Running         2
   ```

> 📖 WSL2 官方文档：https://learn.microsoft.com/zh-cn/windows/wsl/install

### 4. 安装 Docker Desktop

**步骤**:
1. 访问 https://www.docker.com/products/docker-desktop/
2. 下载 Docker Desktop for Windows
3. 运行安装程序
4. **重要**: 在安装选项中勾选 **"Use WSL 2 instead of Hyper-V"**
5. 启动 Docker Desktop
6. 等待 Docker 引擎初始化完成（右下角图标停止闪烁）
7. 验证安装：
   ```bash
   docker --version
   docker-compose --version
   ```

> 💡 **提示**: 首次启动可能需要几分钟，确保虚拟化已在 BIOS 中启用

## 🐧 Linux 环境安装

### 1. 安装 Node.js 20+

**Ubuntu/Debian**:
```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
```

**CentOS/RHEL**:
```bash
curl -fsSL https://rpm.nodesource.com/setup_20.x | sudo bash -
sudo yum install -y nodejs
```

**验证**:
```bash
node --version
npm --version
```

### 2. 安装 .NET 8.0 SDK（可选）

**仅当你需要编译 Luban 配置表时需要**

**Ubuntu/Debian**:
```bash
wget https://packages.microsoft.com/config/ubuntu/$(lsb_release -rs)/packages-microsoft-prod.deb -O packages-microsoft-prod.deb
sudo dpkg -i packages-microsoft-prod.deb
rm packages-microsoft-prod.deb

sudo apt-get update
sudo apt-get install -y dotnet-sdk-8.0
```

**验证**:
```bash
dotnet --version
```

### 3. 安装 Docker

**Ubuntu/Debian**:
```bash
sudo apt-get update
sudo apt-get install -y docker.io docker-compose
sudo systemctl start docker
sudo systemctl enable docker
```

**验证**:
```bash
docker --version
docker-compose --version
```

> 💡 可能需要将用户加入 docker 组：`sudo usermod -aG docker $USER`

## 🍎 macOS 环境安装

### 1. 安装 Node.js 20+

**方式一：Homebrew（推荐）**
```bash
brew install node@20
```

**方式二：官网下载**
1. 访问 https://nodejs.org/
2. 下载 macOS 安装包

**验证**:
```bash
node --version
npm --version
```

### 2. 安装 .NET 8.0 SDK（可选）

**仅当你需要编译 Luban 配置表时需要**

```bash
brew install --cask dotnet-sdk-8.0
```

**验证**:
```bash
dotnet --version
```

### 3. 安装 Docker

**方式一：Docker Desktop（推荐）**
1. 下载：https://www.docker.com/products/docker-desktop/
2. 拖拽到 Applications 文件夹
3. 启动 Docker Desktop

**方式二：OrbStack（轻量替代）**
```bash
brew install orbstack
```

**验证**:
```bash
docker --version
docker-compose --version
```

## 🆘 常见问题

### Q: `dotnet` 命令找不到

**A**: 
1. 确认安装的是 SDK 不是 Runtime
2. 重启终端或 PowerShell
3. 手动添加 PATH: `C:\Program Files\dotnet` (Windows)

### Q: Docker Desktop 无法启动

**A**:
1. 确认已启用 WSL2: `wsl --list --verbose`
2. 确认虚拟化已在 BIOS 中启用
3. 尝试以管理员身份运行 Docker Desktop

### Q: WSL2 安装失败

**A**:
1. 确保 Windows 10 版本 >= 1903 或 Windows 11
2. 以管理员身份运行 PowerShell
3. 执行：`wsl --install -d Ubuntu`

### Q: npm install 失败

**A**:
1. 清理缓存：`npm cache clean --force`
2. 删除 node_modules: `rm -rf node_modules package-lock.json`
3. 重新安装：`npm install`
