# TS-Skynet 首次启动指南

## 🚀 快速开始（3 步）

### 第 1 步：安装依赖

```bash
# 安装根目录依赖
npm install

# 安装 server 目录依赖
cd server && npm install
```

### 第 2 步：启动服务

**任意平台都使用相同命令：**

```bash
npm run quick
```

这将自动：
- ✅ 检查环境
- ✅ 编译 TypeScript → Lua
- ✅ 启动 Skynet 服务

### 第 3 步：完成！

服务启动后，查看日志：

```bash
npm run server:logs
```

## 📁 项目结构

```
tslua/
├── server/
│   ├── src/              # TypeScript 源码
│   ├── dist/lua/         # 编译后的 Lua 代码
│   └── config/           # 配置文件
├── protocols/            # Proto 协议定义
├── tables/               # Luban 配置表
└── docs/                 # 文档
```

---

## ⚙️ 工作原理

1. **编译**：TypeScript 源码通过 TSTL 编译为 Lua
2. **部署**：Lua 代码通过 Docker Volume 挂载到容器
3. **启动**：Skynet 容器加载 Lua 代码运行

---

## 🆘 遇到问题？

### 问题：命令未找到

**解决**：确保已安装依赖

```bash
npm install
cd server && npm install
```

### 问题：Docker 命令失败

**解决**：确保 Docker Desktop 已启动

### 问题：Windows PowerShell 执行策略

**解决**：

```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

或使用 CMD：`start.bat`
