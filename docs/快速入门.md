# TS-Skynet 快速入门指南

> 5 分钟搭建你的第一个 TS-Skynet 服务

---

## 🚀 快速开始（3 步）

### 第 1 步：安装依赖

```bash
# 安装根目录依赖
npm install

# 安装 server 目录依赖
cd server && npm install
```

**预期输出:**
```
added 234 packages in 15s
```

### 第 2 步：启动服务

**任意平台都使用相同命令：**

```bash
npm run quick
```

这将自动：
- ✅ 检查环境
- ✅ 编译 TypeScript → Lua
- ✅ 启动 Skynet 服务

**预期看到:**
```
==================================
  TS-Skynet Hybrid Build Script
==================================
[1/4] Cleaning old builds...
[2/4] Compiling TypeScript to Lua...
✓ Lua compilation complete
[3/4] Compiling TypeScript to JavaScript...
✓ JavaScript compilation complete
[4/4] Copying Lua files to Skynet...
✓ Files copied to skynet/service-ts/
```

### 第 3 步：完成！

服务启动后，查看日志：

```bash
npm run server:logs
```

**预期看到:**
```
[INFO] === Login Service Starting ===
[INFO] Service address: node-service-1706432400000
[INFO] LoginService initializing...
[INFO] === Login Service Ready ===
```

✅ 恭喜！服务运行成功!

## 🔧 环境要求

### 基础环境（所有平台）

- **Node.js**: 20+ ([下载](https://nodejs.org/))
- **npm**: 9+ (随 Node.js 一起安装)

### Windows 用户额外需要

#### 1. .NET 8.0 SDK

**用途**: Luban 配置表编译工具

**下载**: https://dotnet.microsoft.com/download/dotnet/8.0

**验证**: `dotnet --version` 应显示 `8.0.x`

> ⚠️ 注意：需要完整的 SDK，不仅仅是 Runtime

#### 2. Docker Desktop + WSL2

**用途**: 运行 Skynet 生产环境容器

**快速安装**:
```powershell
# 1. 启用 WSL2 (管理员 PowerShell)
wsl --install
wsl --set-default-version 2

# 2. 安装 Docker Desktop
# 下载：https://www.docker.com/products/docker-desktop/
# 安装时勾选 "Use WSL 2 instead of Hyper-V"
```

**验证**:
```bash
docker --version
docker-compose --version
wsl --list --verbose
```

> 📖 详细指南：[Windows Docker 部署指南.md](Windows%20Docker 部署指南.md)

### Linux/macOS

- **Docker**: `sudo apt install docker.io` 或 `brew install docker`
- **.NET 8.0 SDK**: 可选（仅需要编译 Luban 配置表时）

### 安装项目依赖

```bash
# 安装根目录依赖
npm install

# 安装 server 目录依赖
cd server && npm install
```

## 📁 项目结构

```
tslua/
├── server/
│   ├── src/              # TypeScript 源码
│   ├── dist/lua/         # 编译后的 Lua 代码
│   └── config/           # 配置文件
├── protocols/            # Proto 协议定义
├── tables/               # Luban配置表
├── docker/               # Docker 配置
└── docs/                 # 文档
```

---

## ⚙️ 工作原理

1. **编译**: TypeScript 源码通过 TSTL 编译为 Lua
2. **部署**: Lua 代码通过 Docker Volume 挂载到容器
3. **启动**: Skynet 容器加载 Lua 代码运行

## 🆘 遇到问题？

### 问题：命令未找到

**解决**: 确保已安装依赖

```bash
npm install
cd server && npm install
```

### 问题：Docker 命令失败

**解决**: 确保 Docker Desktop 已启动

### 问题：Windows PowerShell 执行策略

**解决**:

```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

或使用 CMD：`start.bat`

### 问题：编译失败 "Cannot find module 'typescript'"

**解决**:

```bash
npm install
```

### 问题：端口被占用

**解决**: 修改 `docker-compose.yml` 中的端口映射：
```yaml
ports:
  - "8889:8888"   # 改为其他端口
```
