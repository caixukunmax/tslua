# 道具配置表示例

## 📋 表格结构设计

### 文件名：`datas/item/道具配置表.xlsx`

#### Sheet1: 道具表 (TbItemConfig)

| 字段名 | 类型 | 说明 | 示例值 |
|--------|------|------|--------|
| id | int | 道具 ID（唯一） | 1001 |
| name | string | 道具名称 | "生命药水" |
| major_type | EMajorType | 主类型 | CONSUMABLES |
| minor_type | EMinorType | 子类型 | FOOD |
| quality | EItemQuality | 品质 | GREEN |
| max_pile_num | int | 最大堆叠数 | 99 |
| icon | string | 图标路径 | "icons/item_1001.png" |
| icon_background | string | 图标背景 | "bg_green" |
| icon_mask | string | 图标遮罩 | "" |
| desc | string | 描述 | "恢复少量生命值" |
| show_order | int | 显示顺序 | 10 |
| effect_type | string | 效果类型 | "heal" |
| effect_value | int | 效果数值 | 50 |
| price | int | 价格 | 100 |
| can_sell | bool | 可否出售 | true |
| obtain_methods | string | 获取途径 | "副本掉落，商店购买" |
| release_date | string | 上线日期 | "2024-01-01" |
